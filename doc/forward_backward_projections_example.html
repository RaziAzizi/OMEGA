forward_backward_projections_example.m

This file is very similar to the gate_main.m and main_PET.m. However, it lacks the possibility for data load (has to be done with one of the other files). Included blocks are SCANNER PROPERTIES, IMAGE PROPERTIES, RAW DATA PROPERTIES, SINOGRAM PROPERTIES, CORRECTIONS, DYNAMIC IMAGING PROPERTIES, MIC PROPERTIES and RECONSTRUCTION PROPERTIES. The last one lacks any reconstruction algorithm related settings. The example shows how to compute, for example, OSEM estimates by using the forward and backward projections.
Forward projection is essentially y = Ax, where y are the measurements (e.g. sinogram), A the system matrix and x the current estimate. Backprojection is then essentially x = ATz, where z is user input vector. In the case of MLEM, z would be z = y/(Ax).
See Computing the forward and/or backward projections for more information.

Forward projection

The forward projection is computed with forward_project.m.

Backward projection

The backprojection is computed with backproject.m.
