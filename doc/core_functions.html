Core functions

gate_main
The complete main-file for GATE data.
main_PET
The complete main-file for non-GATE data.
precompute_data
Precomputation functions (used when options.precompute_lor = true)
load_data
Loads GATE or Inveon data.
form_sinograms
Forms the sinograms from the input raw data.
reconstruction_main
Computes all the necessary steps required for reconstruction and the reconstruction itself.
forward_backward_projections_example
Computes the forward and/or backward projections either with implementation 1 or 3.
custom_prior_test_main
Similar to gate_main except supports custom user-made (gradient-based) priors.
Randoms_variance_reduction
Computes the randoms/scatter variance reduction.
randoms_smoothing
Smoothes the input randoms/scatter data.
normalization_coefficients
Computes the normalization coefficients.
index_maker
Outputs the indices for the specified subset type
get_coordinates
Compute the detector coordinates
set_up_corrections
Loads the necessary correction data.
form_subset_indices
Adjust the data using the indices from index_maker
OMEGA_error_check
Checks the correctness of the input options.
attenuation_correction_factors
Computes the attenuation image for Inveon data.
observation_matrix_formation
Forms the system matrix.
algorithms_char
Outputs the algorithm names and their correct order.
padding
Padds the input image.
Variuous algorithm functions
Computes the algorithms
visualize_pet
Visualization of the output data.
gaussianKernel
Computes 3D Gaussian kernel.
PSFKernel
Computes 3D PSF kernel.
deblur
Computes PSF deblurring.
computeConvolution
Computes PSF blurring with convolution.
