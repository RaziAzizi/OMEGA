This folder contains the mex-files needed to load list-mode data from Siemens Biograph mCT and Vision PET/CT devices.

list2sinogram corresponds to mCT and visionToSinogram to Vision. mexw64 refers to the (64-bit) Windows binary and mexa64 to the (64-bit) Linux binary.

Windows version was built with MATLAB R2015a using Microsoft Windows SDK 7.1.
Linux version was built with MATLAB R2015b using G++ 9.3.0 on Ubuntu 20.04.
